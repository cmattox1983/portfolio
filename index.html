Here are some of my projects

  • Netflix Clone Project

    HTML, Tailwind CSS, React, TMDB API

    A fully responsive Netflix clone built with React and Tailwind CSS, powered by the TMDB API for real-time movie data. Users can browse trending titles, explore categories, and view detailed information in a clean, modern interface designed to mirror a real streaming experience.

  • Skinstric AI Internship

    HTML, Tailwind CSS, Next, React, Vercel, Figma, Base64

    An AI-powered web app where users can upload or capture a selfie and receive predicted demographic insights such as age, gender, and ethnicity. Built with Next.js and React, the project integrates facial recognition APIs, real-time image processing, and dynamic UI updates to deliver a seamless, interactive experience.

  • Summarist Virtual Internship

    HTML, CSS, Next, React Icons, Redux, Firebase, Stripe, Vercel

    A full-stack book summary platform where users can create free or premium accounts, browse titles via API integration, and add books to their cart to access audio versions and written summaries. Built with Next.js, Redux, Firebase, and Stripe, the project focuses on authentication, state management, and seamless user experience.

  • Ultraverse Project

    HTML, CSS, React, Bootstrap, Owl Carousel

    In this internship project, I transformed a fully static book website into a dynamic, API-driven application. I integrated multiple endpoints to render books by category, fetch individual books by ID, and display them using Owl Carousel, focusing on clean data flow and real-time content rendering.

  • Posterize Movie Project

    HTML, CSS, React, IMDb API

    A React-based movie search application powered by the IMDb API, allowing users to search by title and instantly access posters, summaries, ratings, and official IMDb links. Features include dynamic rendering, real-time API integration, and sorting by newest, oldest, or highest-rated results for a smooth browsing experience.

  • Library Project

    HTML, CSS, React

    A mock e-commerce library website built with React, featuring dynamic routing, shopping cart functionality, and sorting capabilities. The project focuses on state management, reusable components, and creating a smooth, interactive browsing and checkout experience.

  • Treact Project

    HTML, CSS, JavaScript

    My first project focused on building a fully responsive landing page using HTML, CSS, and JavaScript. It emphasized strong layout structure, clean styling, and interactive elements, showcasing my foundation in frontend development and attention to design detail.
